# JWT-1 — Next.js Authentication Demo

Prosty projekt pokazujący, jak zbudować bezpieczne logowanie w Next.js z użyciem:

- Access Token 
- Refresh Token 
- httpOnly Secure Cookies
- Proxy (Middleware) chroniącej strony
- Minimalnego UI (Login / Profile / Home)

Projekt edukacyjny — do nauki podstaw JWT i autoryzacji.
